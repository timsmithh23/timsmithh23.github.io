# VLAN Segmentation and Routing

This project documents how I segmented my homelab network using VLANs and set up inter-VLAN routing.

## Objective
- Isolate traffic between servers, management, and IoT devices

## Setup
- Layer 3 switch configuration
- Proxmox VM network setup

## Outcome
- Network is segmented
- VMs on different VLANs can communicate securely via routing
